How to contribute¶

If after using or testing protopipe you find that something is missing, wrong, or simply doesn’t do what you expect, you are invited to join the development effort.

This section contains,

  • information on how the project is organized (The repository),

  • instructions on how to start adding your contribution (Instructions),

  • what you can check before adding it (Code and performance checks).
