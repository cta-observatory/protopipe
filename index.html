Welcome to protopipe’s documentation!#

Protopipe is a pipeline prototype for the Cherenkov Telescope Array (CTA) based on the ctapipe and pyirf libraries among other third-party Python packages.

Warning

This is not yet stable code, so expect large and rapid changes.
